**Info**
Este es un proyecto para hacer una pagina web dinámica siendo un manejador de contenido. 
Se creo en la carrera de Ingenieria en Sistemas de Información en la Universidad de Sonora.
Uso básico de CRUD con PHP, HTML Y CSS.
Cuenta con la temática de un Cine, con la idea de mostrar las funciones.

**Uso**
El Administrador sube las peliculas con la vista de administrador y los controles visibles solo para el administrador.
La pagina mostrará las peliculas subidas por el administrador, para que el publico pueda ver que peliculas están disponibles.

**Configuración**
Se instala el servidor apache con XAMPP, se crea una base de datos llamada "cinemafilms" y se importa el script que se encuentra en la carpeta raiz del proyecto a la base de datos.
